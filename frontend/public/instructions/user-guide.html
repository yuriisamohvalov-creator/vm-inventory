Руководство пользователя VM Inventory

Документ для ежедневной работы в системе учета виртуальных машин.

1. Вход

  1. Откройте главную страницу приложения.
  2. Введите логин и пароль.
  3. Нажмите «Войти».

2. Виртуальные машины

  • Создание и редактирование ВМ (при наличии прав).
  • Валидация IP и предупреждение о дубликатах.
  • Назначение ресурсов (CPU/RAM/Disk), тегов и привязки к ИС.

3. Общие пулы

Раздел для группировки ВМ. Система контролирует совместимость ВМ по бизнес-ограничениям.

4. Отчеты

Иерархический отчет доступен для просмотра и экспорта в PDF/XLSX/JSON.

5. Подсказки

  • Если нет прав на изменение — отображается режим только чтения.
  • При ошибке авторизации выполните повторный вход.
