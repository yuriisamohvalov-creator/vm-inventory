Руководство администратора VM Inventory

Документ по развертыванию, администрированию пользователей и подготовке к Kubernetes.

1. Развертывание (Docker/Podman)

cp .env.example .env
docker compose up -d --build
# или
podman compose up -d --build

2. Создание пользователей

  1. Войдите под администратором.
  2. Откройте «Администрирование» → вкладку пользователей.
  3. Создайте учетную запись и задайте роль.

3. Роли

  • administrator — полный доступ.
  • analyst — чтение и экспорт.

4. Эксплуатация

docker compose exec backend python manage.py migrate
docker compose exec db pg_dump -U vminventory vminventory > backup.sql

5. Kubernetes readiness

  • ConfigMap/Secret для конфигурации.
  • Deployments для backend/frontend/nginx.
  • Ingress + TLS + health probes.
